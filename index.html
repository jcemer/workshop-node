Neste workshop você vai aprender a desenvolver aplicações utilizando node.js, uma plataforma para aplicações de rede altamente escaláveis. Começando pelo básico, desde instalação a como utilizar o NPM e o sistema de módulos, até conceitos avançados como streams, bibliotecas de controle de fluxo, e criação de servers utilizando HTTP e websockets.

Não é necessário ter experiência prévia com node.js, mas bons conhecimentos de javascript são necessários. Se você quer entender melhor os fundamentos de programação assíncrona e como utilizá-los para criar aplicações web mais eficientes, este curso é para você!

O que você vai aprender

  1. Introdução

    Conheça as peculiariedades do Javascript no servidor, o engine V8 e seu papel no Node.js.

  2. Por que Node.js?

    O Node.js introduz conceitos particulares de execução assíncrona. Você irá descobrir isto e muito mais e entender o porquê e quando usar Node.js.

  3. Sistema de módulos

    Incorporando as melhores práticas de modularização, o NPM é o gerenciador de pacotes padrão do Node.js.

    Entenda o sistema de módulos, como organizar seu código e como distribuir seus próprios pacotes.

  4. Construindo uma aplicação

    Aprenda a contruir uma aplicação HTTP simples e como evoluí-la para o framework Express.

  5. Streams

    O conceito de streams é bastante presente e fundamental para as características de performance do Node.js.

    Entenda o que é, como aplicar em seu código e o quanto é importante manter seus dados em fluxo constante.

  6. Aplicações em tempo real

    Aprenda a utilizar Websockets para transmitir dados em tempo real para navegadores.

  7. Guest talk

    Convidado especial apresenta aplicações da tecnologia no mercado e sua experiência com a plataforma em um ambiente de produção.

É para você!

O conteúdo deste workshop abrange do básico ao avançado da tecnologia por trás do Node.js.

É imprescindível conhecimentos de programação e em especial o básico da linguagem Javascript. Aconselhamos a leitura do JavaScript Garden para reforçar seus conhecimentos.

Você precisa apenas ir munido do seu entusiasmo, não necessário, porém não restritivo, que você traga seu laptop ou tablet.
